## Overview

A responsive recipe page featuring a simple omelette recipe with preparation time, ingredients, step-by-step instructions, and nutritional information. The page is designed with semantic HTML and custom CSS styling.

## Getting Started

To view this project locally:

1. Clone the repository:
```bash
git clone <repository-url>
```

2. Navigate to the project directory:
```bash
cd "FM Recipe Page"
```

3. Open `index.html` in your browser:
```bash
open index.html
```

Or use a local development server for better experience:
```bash
# Using Python
python -m http.server 8000

# Using Node.js http-server
npx http-server
```
